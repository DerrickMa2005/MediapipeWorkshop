# MediapipeWorkshop

## Virtual Environment

While optional, we recommend setting up a virtual environment before running this project to prevent dependencies from this project from clashing with dependencies from your other projects.

You can set up a virtual environment using [Venv](https://packaging.python.org/en/latest/guides/installing-using-pip-and-virtual-environments/).

## How to Run

**Step 1**

Install the dependencies within dependencies.txt

```
pip install -r requirements.txt
```

**Step 2**

Run the python file

```
python mediapipeWorkshop.py
```

**Step 3**

Close out of the program by pressing q or pressing Ctrl+C in the terminal.
